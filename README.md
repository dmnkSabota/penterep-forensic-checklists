# penterep-forensic-checklists

A digital forensic checklist toolkit compliant with **NIST SP 800-86** (*Guide to Integrating Forensic Techniques into Incident Response*) and **ISO/IEC 27037:2012** (*Guidelines for identification, collection, acquisition, and preservation of digital evidence*).

## Overview

This repository provides structured, step-by-step checklists for common digital forensic scenarios. Each scenario is broken down into individual, clearly named steps so that forensic analysts can follow a repeatable, auditable process.

## Scenarios

| Folder | Description |
|--------|-------------|
| [`chain-of-custody/`](chain-of-custody/) | Procedures for maintaining an unbroken chain of custody for digital evidence |
| [`malware-analysis/`](malware-analysis/) | Workflow for safe acquisition and analysis of malware samples |
| [`photo-recovery/`](photo-recovery/) | End-to-end process for recovering and validating photographs from digital media |

## Standards Compliance

- **NIST SP 800-86** – Guide to Integrating Forensic Techniques into Incident Response
- **ISO/IEC 27037:2012** – Guidelines for identification, collection, acquisition, and preservation of digital evidence
